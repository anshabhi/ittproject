About Facial Attendance System

This system has been designed to automate the process of taking attendance, resulting in greater efficiency and saving time for more useful purpose. The project uses cutting edge technologies to detect faces quickly. The frontend is built using Bootstrap (HTML5, CSS3) and backend has been built using Python. CGI is used for interfacing the frontend and backend.

Advantages of using this Online Attendance Management System:

  • Very Efficient
  • No Proxy
  • Automatic Updation of Attendance
  • Time Saving
